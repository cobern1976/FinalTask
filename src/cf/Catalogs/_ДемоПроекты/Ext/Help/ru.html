Справочник предназначен для демонстрации примера использования ряда подсистем.

В подсистеме Базовая функциональность справочник демонстрирует установку параметра сеанса и формирование заголовка главного окна программы. См. команду Сделать основным контекстного меню списка проектов.

В подсистеме Бизнес-процессы и задачи элементы справочника используются в качестве объектов адресации задач (при ролевой адресации).

В подсистеме Настройка порядка элементов элементы справочника могут быть упорядочены пользователем.

В подсистеме Префиксация объектов демонстрируется переназначение номера справочника при смене организации.

В подсистеме Присоединенные файлы элементы справочника используются в качестве объектов-владельцев файлов вложений. Реквизит РегламентПроекта демонстрирует форму выбора присоединенных файлов. В форме списка добавлена колонка со скрепкой - признаком наличия вложений.

В подсистеме Управление доступом показывает пример типичного использования логики ограничения по одному виду доступа.

  • Список ограничивается по одному виду доступа:
    • Организации (проверяется значение реквизита Организация).
